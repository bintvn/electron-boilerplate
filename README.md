# Electron Boilerplate

A modern **Electron + React + TailwindCSS + TypeScript** boilerplate for building cross-platform desktop applications.

***Currently Support for Windows Only***

---

## 🚀 Features

- ⚡ Electron (Desktop App Framework)
- ⚛️ React (UI Layer)
- 🎨 TailwindCSS (Styling)
- 🟦 TypeScript (Type Safety)
- 🔥 Vite (Fast Development Build Tool)
- 📦 Ready for production build (electron-builder)

---

## 📦 Requirements

Before starting, make sure you have:

- Node.js (>= 18 recommended)
- npm
- Windows: Enable **Developer Mode**

---

### Enable Developer Mode (Windows)

Settings → System → For Developers → Developer Mode → ON

---

## 🛠 Installation

Install all dependencies:

```bash
npm install
```

Run the app in development mode:

```bash
npm run dev
```

Build the application for production:

```bash
npm run build:release
```
